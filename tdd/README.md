# TDD skill

Языконезависимый агентский навык для работы по TDD.

Структура:

- `SKILL.md` — основной навык;
- `guides/behavior-tests.md` — поведенческие тесты;
- `guides/vertical-slices.md` — вертикальные срезы;
- `guides/deep-modules.md` — глубокие модули;
- `guides/interface-design.md` — дизайн интерфейсов;
- `guides/mocking.md` — mock, fake, stub;
- `guides/refactoring.md` — рефакторинг после GREEN;
- `guides/checklist.md` — чеклист TDD-цикла.

Примеры намеренно написаны на псевдокоде, чтобы навык не был привязан к конкретному языку программирования или тестовому фреймворку.

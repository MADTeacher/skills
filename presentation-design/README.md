# mad-presentation-design

Локальный агентский навык для создания и доработки презентаций.

## Что делает

- HTML-презентации и системы слайдов.
- Редактируемые PPTX из HTML, написанного с ограничениями PowerPoint.
- PDF-экспорт из HTML-презентаций.
- PNG-снимки слайдов.
- Проверяемый художественный SVG-путь для native SVG-слайдов, векторных
  обложек, сложных сцен, схем, орнаментов и декоративных ассетов.
- Экспертная проверка презентаций и сравнение вариантов слайдов.

## Основной процесс

1. Уточнить аудиторию, цель, логику рассказа, объем и форматы сдачи.
2. Собрать бренд и исходные материалы.
3. Сделать 2-слайдовое направление: обложка и плотный контентный слайд.
4. Собрать систему слайдов.
5. Проверить HTML в браузере.
6. Экспортировать нужные форматы.

Подробности лежат в `references/workflow.md`.

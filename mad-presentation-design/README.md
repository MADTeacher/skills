# mad-presentation-design

Локальный агентский навык для создания и доработки презентаций.

## Что делает

- HTML-презентации и deck-системы.
- Редактируемые PPTX из constrained HTML.
- PDF-экспорт из HTML-деков.
- PNG-снимки слайдов.
- Ограниченный SVG-путь для специально подготовленных векторных слайдов.
- Дизайн-ревью презентаций и сравнение вариантов слайдов.

## Основной процесс

1. Уточнить аудиторию, цель, storyline, объем и форматы сдачи.
2. Собрать бренд и исходные материалы.
3. Сделать 2-слайдовое направление: обложка и плотный контентный слайд.
4. Собрать систему слайдов.
5. Проверить HTML в браузере.
6. Экспортировать нужные форматы.

Подробности лежат в `references/workflow.md`.

# Процесс художественного SVG

Используйте этот файл, когда нужно создать сцену с нуля, усилить слабую композицию или превратить расплывчатый запрос в план SVG.

## Бриф

Снимите бриф коротко. Если пользователь не дал ответ, выберите безопасное значение и отметьте это в сдаче.

- Сюжет и главный объект сцены.
- Формат: горизонтальный, вертикальный, квадратный, эмблема или обложка.
- Стиль: строгий, мягкий, технический, сказочный, плакатный, минималистичный, декоративный.
- Настроение: спокойное, напряжённое, тёплое, холодное, праздничное, таинственное.
- Палитра: тёмная, светлая, монохромная, контрастная, пастельная, ограниченная.
- Ограничения: без внешних картинок, без скриптов, без мелкого текста, без тяжёлых фильтров.
- Место использования: сайт, книга, презентация, репозиторий, иконка, печать.
- Уровень детализации: быстрый эскиз, средняя иллюстрация, сложный постер.

## План сцены

Сначала опишите для себя сцену как слои. Затем пишите SVG.

1. Холст: `viewBox`, фон, безопасные размеры, `title` и `desc`.
2. Крупные массы: силуэты, диагонали, горизонт, рамки, главный объект.
3. Средний план: повторяемые формы, вторичные объекты, орнаменты, архитектура.
4. Свет: источники, тени, мягкие градиенты, ореолы, отражения.
5. Фактура: тонкие линии, шум через SVG-примитивы, паттерны, полупрозрачные слои.
6. Акценты: маленькие контрастные детали, блики, подписи, маркировка.
7. Проверка: читается ли сцена в маленьком размере, нет ли случайной грязи, не спорят ли детали с фокусом.

## Композиционные решения

- Для постера используйте один сильный фокус и вторичный ритм вокруг него.
- Для сцены держите три плана: передний, средний, дальний. Это даёт глубину без фотореализма.
- Для эмблемы используйте крупный силуэт, внутренний знак, ограниченную палитру и чистый контур.
- Для обложки оставьте место под текст. Проверяйте читаемость на PNG-рендере.
- Для орнамента стройте модуль и повтор, затем добавляйте один сбой ритма как живую деталь.

## Значения по умолчанию

| Задача | `viewBox` | Детализация | Цвет |
|---|---:|---|---|
| Горизонтальная сцена | `0 0 1200 800` | средняя или высокая | 5–8 основных цветов |
| Вертикальный постер | `0 0 800 1200` | высокая | 5–9 основных цветов |
| Квадратная иллюстрация | `0 0 1024 1024` | средняя | 4–7 основных цветов |
| Эмблема | `0 0 512 512` | низкая или средняя | 2–5 основных цветов |
| Обложка | `0 0 1600 2400` | средняя | 5–8 основных цветов |

## Цикл правки

1. Создайте SVG по плану слоёв.
2. Запустите `node .agents/skills/artistic-svg-director/scripts/svg_pipeline.mjs audit <file.svg> <preview.png>`.
3. Откройте PNG и оцените картинку по `references/visual-review-rubric.md`.
4. Исправьте SVG: фокус, контраст, читаемость, глубину, цвет, лишний шум.
5. Повторите цикл. Для сложной сцены хватит двух проходов, если пользователь не просит дальше.

## Когда уточнять бриф

- Размер или место использования влияют на формат файла.
- Пользователь просит точное сходство с человеком, логотипом или защищённым персонажем.
- Пользователь требует внешний ресурс, который нельзя безопасно встроить.
- Нужен стиль живого художника и запрос нельзя выполнить буквально.
- Нужна анимация, а среда публикации запрещает SMIL или CSS-анимацию.
